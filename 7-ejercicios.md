# Respuestas y justificación – Elección de bases de datos

## Daniel Cotarelo Calderón

## Ejercicio 1 – Base de datos relacional (SQL)

**Respuesta:** Base de datos relacional (SQL).

**Justificación:**  
Existen relaciones bien definidas entre las entidades y se requiere consistencia fuerte en todo momento.  
El sistema es transaccional y necesita cumplir las propiedades ACID, con un modelo de datos estable y estructurado.

---

## Ejercicio 2 – Base de datos Clave–Valor (NoSQL)

**Respuesta:** Base de datos Clave–Valor.

**Justificación:**  
El acceso se realiza directamente por clave, sin relaciones entre datos.  
Se prioriza la velocidad y los datos son temporales, sin necesidad de consultas complejas.

---

## Ejercicio 3 – Base de datos Columnar (NoSQL)

**Respuesta:** Base de datos columnar.

**Justificación:**  
Se maneja un volumen muy alto de datos y predominan las lecturas analíticas.  
Las consultas están orientadas a agregaciones y patrones globales, no a registros individuales.

---

## Ejercicio 4 – Base de datos Documental (NoSQL)

**Respuesta:** Base de datos documental.

**Justificación:**  
Las entidades presentan estructuras variables y el modelo cambia con frecuencia.  
Cada entidad es autocontenida y se requieren consultas flexibles por distintos campos.

---

## Ejercicio 5 – Base de datos relacional (SQL)

**Respuesta:** Base de datos relacional (SQL).

**Justificación:**  
Es un sistema altamente transaccional donde la consistencia es crítica.  
No se puede aceptar consistencia eventual y se requiere atomicidad y durabilidad.

---

## Ejercicio 6 – Base de datos de Grafos (NoSQL)

**Respuesta:** Base de datos de grafos.

**Justificación:**  
El valor del sistema está en las relaciones entre entidades.  
Se necesitan consultas basadas en recorridos y conexiones indirectas que en SQL serían costosas.

---

## Ejercicio 7 – Base de datos Vectorial (NoSQL)

**Respuesta:** Base de datos vectorial.

**Justificación:**  
La búsqueda se basa en similitud semántica y comparaciones aproximadas.  
Se utilizan representaciones vectoriales integradas con modelos de IA.

---

## Ejercicio 8 – Base de datos relacional (SQL)

**Respuesta:** Base de datos relacional (SQL).

**Justificación:**  
El modelo de datos es simple, estable y con volumen controlado.  
Las relaciones son claras y no se justifica la complejidad de una solución NoSQL.
